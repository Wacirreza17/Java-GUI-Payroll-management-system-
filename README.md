# Java-GUI-Payroll-management-system-

This is a very basic implimentation of the oop principles. 
worker folder is an user defined package. This folder contain the EMP file which stores user data to be used to validate acess and maintaining proper security.

Start.java file contains the main class this where the code is executed from. 

LoginPage,.java creates the initial verification GUI requiring employee Id and their respective passwords.

DashboardPage.java contains the arithmatic operation done to calculate or deduct salaries. It also contains exception handeling via try and catch methods. This file also contains the largest amount of action performed logics as it acts as a bridge between the loginPage and Start.java.
